# Project Starter

Чистая заготовка для нового проекта на `Gulp`.

## Команды

```bash
npm install
npm run dev
npm run build
```

## Базовая структура

- `src/pages` — страницы
- `src/templates` — layout-шаблоны
- `src/blocks` — блоки и компоненты
- `src/scss` — глобальные стили
- `src/js` — JS-точки входа
- `src/static/images` — изображения
- `dist` — результат сборки
